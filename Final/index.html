About Me

Born in 1997, I was raised as the youngest of 3 by my two parents. In my spare time I like to play video games (Currently Destiny 2 and PLAYERUNKOWN'S BATTLEGROUND), run a Discord channel, volunteer for community service, and immerse myself in the world of programming. My favorite languages are HTML/CSS and C++. I'm a fan of seeing immediate progress on the work that I do. I attend NYU's Tandon School of Engineering as a Computer Science majore and a double minor in Integrated Digital Media and Game Programming. I currently work as a Teaching Assistant and an Administrator for the General Engineering Department and a member of NYU's Society of Hispanic Professional Engineers.

Education

New York University 2015-2019

Introduction to Programming & Problem SolvingA-

This course introduces problem solving and computer programming. The course covers fundamentals of computer programming and its underlying principles using the Python programming language.

Introduction to Engineering & DesignA

This course introduces selected aspects of the history, philosophy, methodology, tools and contemporary topics in engineering. Also included are basic engineering experimentation and data analysis, a team design project and analysis and presentation of engineering data and designs.

Introduction to Computer ScienceB+

This is a course that introduces computer-science majors to several subdisciplines in the computer-science field. The course is built around the theme that computer science is the study of algorithms and includes much more than programming. The course introduces hardware, virtual machines, software, applications and social issues in computing.

Object Oriented ProgrammingB-

This course teaches object-oriented programming in C++ through pointers, dynamic memory allocation and recursion, Classes and objects, Access and the interface to relationships of classes, Polymorphism, Inheritance, and templates.

Computer Architecture & OrganizationA-

This course covers a top-down approach to computer design. Topics: Computer architecture, introduction to assembly language programming and machine language set design. Computer organization, logical modules; CPU, memory and I/O units. Instruction cycles, the datapath and control unit. Hardwiring and microprogramming. The memory subsystem and timing. I/O interface, interrupts, programmed I/O and DMA. Introduction to pipelining and memory hierarchies.
